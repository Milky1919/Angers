# アンダーグラウンド画像検索エンジン構築計画 - "Angers"

このプロジェクトでは、既存のオープンソース検索エンジン（SearXNGベースを推奨）をカスタマイズし、Dockerを使用して独自の「アングラ系」画像検索サイトを構築します。

## 1. コンセプトと機能要件
- **画像特化**: デフォルトで画像検索を表示。
- **無修正・フィルタなし**: セーフサーチを無効化し、あらゆるソースからの結果を表示。
- **アングラ感**: UIデザインをダークモード、ハッカー/サイバーパンク風にカスタマイズ。
- **独自性**: Googleがインデックスしないサイト（Deep Webや特定の画像掲示板など）も対象にできるよう、カスタムエンジンの追加や設定を行う。

## 2. 技術スタック
- **ベースソフトウェア**: [SearXNG](https://github.com/searxng/searxng) (メタ検索エンジン)
  - 理由: 高度なカスタマイズが可能、Docker対応、プライバシー重視、多様な検索エンジン（画像サイト含む）を統合可能。
- **インフラ**: Docker, Docker Compose
- **データベース/キャッシュ**: Redis (SearXNGのキャッシュ用)

## 3. 実装ステップ

### Step 1: 基本環境の構築
- `docker-compose.yml` の作成
- SearXNGとRedisの連携設定

### Step 2: 設定のカスタマイズ (settings.yml)
- **検索設定**:
  - `safe_search: 0` (フィルターなし)
  - デフォルトカテゴリを `images` に固定
- **エンジンの有効化**:
  - 一般的な検索エンジンに加え、特殊な画像サイトや掲示板（Dribbble, DeviantArt, Flickr, その他設定可能なもの）を有効化。
  - 必要であればカスタムスクレイパーを追加（要Pythonコーディング）。

### Step 3: UI/デザインの改造 ("アングラ化")
- SearXNGの静的ファイル（CSS/HTML）をオーバーライド。
- **デザインテーマ**:
  - 背景: 黒または濃いグレー
  - アクセントカラー: ネオングリーン (Matrix風) または ネオンパープル/レッド
  - フォント: モノスペースやターミナルフォント
  - アニメーション: グリッチエフェクトなどをCSSで追加

### Step 4: 動作確認と調整
- ローカルでの起動 (`docker-compose up`)
- 検索精度の確認
- UIの調整

## 4. (オプション) 独自クローラーの追加
もしメタ検索だけでは不十分な場合、特定のサイトを巡回してインデックスを作成するサブシステム（例: Scrapy + Meilisearch）を追加検討します。
